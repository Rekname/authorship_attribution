Десктопное приложение для определения автора анонимного текста методами **стилометрии** и **машинного обучения**. Поддерживаются русский и английский языки. Графический интерфейс построен на PyQt6.

## Возможности

- **Анализ** — загрузка анонимного текста и предсказание автора несколькими классификаторами
- **Авторы** — просмотр стилометрического профиля авторов из корпуса: богатство словаря, длина слов и предложений, доля пунктуации и др
- **Обучение** — обучение модели на корпусе текстов с индикатором прогресса и сравнением точности классификаторов на отложенной выборке

## Требования

- Python **3.12**
- ОС: Linux / Windows / macOS
- [uv](https://docs.astral.sh/uv/) — рекомендуемый менеджер пакетов

## Установка

### Вариант 1 — через uv (рекомендуется)

```bash
git clone <repo-url>
cd authorship_attribution
uv sync
uv run python -m spacy download en_core_web_sm
uv run python -m spacy download ru_core_news_sm
```

### Вариант 2 — через pip

```bash
git clone <repo-url>
cd authorship_attribution
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m spacy download en_core_web_sm
python -m spacy download ru_core_news_sm
```

## Подготовка данных

Корпус не входит в репозиторий. Структура каталога `data/`:

```
data/
├── english/
│   └── <Имя_Автора>/
│       ├── <Книга 1>.txt
│       └── <Книга 2>.txt
├── russian/
│   └── <Имя_Автора>/
│       ├── <Книга 3>.txt
│       └── <Книга 4>.txt
└── test_samples/ 
    └── <Книга 5>.txt
    └── <Книга 6>.txt
```

- Каждый автор — отдельная подпапка, имена файлов произвольные `*.txt`

## Использование интерфейса

Приложение открывается с тремя вкладками

### Вкладка «Обучение»

1. Нажмите кнопку обучения — корпус из `data/` загружается автоматически
2. Прогресс отображается в индикаторе
3. По завершении показывается таблица с точностью каждого классификатора
4. Модель сохраняется в `models/classifier.joblib`, конфигурация в `models/training_config.json`, и подгружается при следующем запуске

### Вкладка «Анализ»

1. Загрузите файл с анонимным текстом
2. Выберите классификатор
3. Приложение покажет наиболее вероятного автора и распределение
   вероятностей по всем авторам в виде графика
   
### Вкладка «Авторы»

1. Выберите автора из списка, чтобы увидеть его стилевой профиль
